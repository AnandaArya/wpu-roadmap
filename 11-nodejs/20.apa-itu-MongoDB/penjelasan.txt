MongoDB?
- MongoDB is a document database with the scalabity and flexibility that you want with querying and indexing that you need
- MongoDB adlaah sebuah database berbasis dokumen yang memiliki skalabilitas dan fleksibilitas yang kita inginkan dan juga
  memiliki fitur query dan indexing yang kita butuhkan

Konsep Embedded Document (Denormalization)
{
    "book_id": 1,
    "isbn": "971238213",
    "title": "ELoquent Javascript, Second edition",
    "author: "Marijn Haverbeke",
},
{
    "book_id": 2,
    "isbn": "97123823",
    "title": "Learning Javascript Design Pattern",
    "author: "Addy Osmani",
    "publisher": {
        "publisher_id" : 21,
        "name": "o Reilly Media",
    } // ini disebut dengan Embedded Document, karena ini juga disebut dengan Schemaless
},

Kalau kita ingin menerapkan ini di SQL, tentu saja kita harus membuat tabel baru, jadi kita punya table buku, dan publisher
dimana kita akan punya relationshipnya, kita harus punya foreign key di tabel buku yang mengarah ke primary key di table publisher.
Tapi untuk NoSQL tidak seperti itu kita cukup menggunakan Embedded Document
